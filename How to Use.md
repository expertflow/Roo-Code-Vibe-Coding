# How to Use

Copy the two files (Install-RooCode.bat + Install-RooCode.ps1) to any machine
Double-click Install-RooCode.bat
If Windows SmartScreen appears, click More info → Run anyway
Restart the IDE(s) after installation completes
Open Roo Code from the sidebar and select a profile from the dropdown
Note: For Gemini profiles, the user must also run gcloud auth application-default login once to authenticate with Google Cloud.
